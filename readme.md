# jdk-manager

## Install

```console
$ npm install jdk-manager -g
```

Init jdk-manager
```console
$ jdkm --init
```

Update (after java update)
```
$ jdkm --update
```

While install or update, the cli ask for alias for all jdk folder ( auto-detect ).

Set jdk
```
$ jdkm --set 8 //'8' is a alias for a jdk folder (like jdk1.8.1_191/)
```
