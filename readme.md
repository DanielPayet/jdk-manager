# jdk-manager

## Install

```console
$ npm install jdk-manager -g
```

Init jdk-manager
```console
$ jdkm --init
```

Update (after java update)
```
$ jdkm --update
```

While install or update, the cli will ask you to give alias to jdk version (auto-detect).

Set jdk
```
$ jdkm --set 8 //'8' is a alias for a jdk folder (like jdk1.8.1_191/)
```

Need help ? 
```
$ jdkm -h
```
